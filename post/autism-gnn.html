2025-06 | GNN · fMRI · ASD

Diagnosis of Autism based on Graph Neural Networks

基于图神经网络的孤独症谱系障碍（ASD）诊断分类研究。利用 ABIDE 数据集中的 fMRI 脑功能连接特征，结合被试表型信息构建人群图，采用 Chebyshev 多项式近似 GCN 和自注意力增强的 GAT 模型进行二分类（ASD vs 健康对照）。

核心原理

核心思想

将每个被试视为图中的一个节点，特征为 fMRI 功能连接向量，边由表型亲和图与高斯核距离图的逐元素乘积决定，然后用 GNN 进行节点分类。

图构建

  1. 表型亲和图：基于 SEX 和 SITE_ID 构建二值邻接矩阵（同性别 + 同站点 = 1）
  2. 高斯核距离图：基于特征的相关性距离 + 高斯核函数
  3. 最终邻接矩阵 = 表型亲和图 ⊙ 高斯核距离图（逐元素相乘）
  4. Chebyshev 多项式展开：4 阶多项式近似，生成 4 个稀疏支持矩阵

特征工程

  1. 从 _ho_correlation.mat 加载功能连接矩阵
  2. Fisher z 变换（arctanh）后提取上三角元素
  3. 删除全零列
  4. RidgeClassifier + RFE 特征选择至 2000 维
  5. 行归一化

模型变体

GCN

Input(613 × 2000)
  → GraphConvolution(2000, 256) + ReLU + Dropout(0.3)
  → GraphConvolution(256, 96)
  → MLP: 96 → 64 → 16 → 2text

GAT（自注意力增强）

Input(613 × 2000)
  → GraphConvolution(2000, 256) + ReLU + Dropout(0.3)
  → Attention(scaled_dot_product) + Squeeze
  → GraphConvolution(256, 96)
  → MLP: 96 → 64 → 16 → 2text

CAMV_GCN / CAMV_GAT

组合多注意力视图架构（设计但未完全启用）。设计了双视图（FC + HOFC）架构，当前仅启用 FC 单视图分支。

注意力机制

models/attention.py 提供了 4 种评分函数：

函数 说明 Accuracy AUC
dot_product 点积注意力 — —
scaled_dot_product 缩放点积注意力（最终选用） 74.80% 78.12%
mlp MLP 注意力（concat 方式） 72.36% 76.58%
bi_linear 双线性注意力（general dot 方式） 73.98% 76.66%

数据

本项目使用 ABIDE（Autism Brain Imaging Data Exchange）数据集，预处理管道为 C-PAC，配置为带通滤波、无全局信号回归（filt_noglobal）。

  • 被试数量：871 人（筛选后 613 人，年龄 ≤ 18）
  • 脑区划分：Harvard-Oxford atlas（111 个 ROI）
  • 特征来源：ROI 时间序列 → 功能连接矩阵（correlation）→ Fisher z 变换 → 上三角提取
  • 表型信息：DX_GROUP（诊断标签）、SEX（性别）、SITE_ID（采集站点）、AGE_AT_SCAN（年龄）、FIQ（智商）

使用方法

训练

# 训练 CAMV_GCN
python train.py

# 训练 CAMV_GAT
python train_gat.py

# 带 TensorBoard 的 baseline 训练
python train_final_baseline.py

# 标准 GAT demo
python train_GAT_demo.pybash

测试

# 加载预训练模型，逐样本预测
python modelTest.py

# GAT 模型测试
python test.pybash

超参数

参数 默认值 说明
--learning_rate 0.003 学习率
--epochs 100 训练轮数
--hidden 16 隐藏层维度
--dropout 0.3 Dropout 率
--weight_decay 5e-4 权重衰减
--embed_dim 256 嵌入维度
--hidden_dim 256 注意力隐藏维度
--head 1 注意力头数
